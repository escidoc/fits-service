TME Service

The TME (Technical Metadata Extractor) Service is currently a wrapper for the File Information Tool Set (FITS). FITS is developed by the Harvard University Library Office for Information Systems for use in its Digital Repository Service (DRS). The TME Service is build by FIZ Karlsruhe for use in the eSciDoc Project.

Usage

examine
Examine a given file and return technical metadata XML.
path
The locator of the file to examine. This may be a local path in the servers filesystem or a HTTP URL.
returns
XML document containing technical metadata (FITS output XML).
